# Local Uncensored AI - Tudo em Um

## 🎯 O que este repositório contém

Repositório completo para rodar **Stable Diffusion uncensored** localmente com o máximo de funcionalidades possíveis.

### Funcionalidades incluídas:

- **Geração de imagens NSFW sem restrições** (mulheres magras, seios pequenos, bigodinho, etc.)
- **Text-to-Image, Image-to-Image, Inpainting, Outpainting**
- **ControlNet** (pose, depth, openpose, etc.)
- **LoRAs, Embeddings, Hypernetworks**
- **Face Restore, Upscale (4x, 8x)**
- **Video generation** (AnimateDiff, SVD)
- **Training** de LoRAs próprios
- **WebUI Forge + ComfyUI** setups
- Prompts otimizados para seu estilo favorito
